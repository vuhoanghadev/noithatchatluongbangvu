Hướng dẫn tạo Favicon cho Nội Thất Bàng Vũ

Vấn đề: Favicon SVG không hiển thị trên kết quả tìm kiếm Google. Chúng ta cần tạo favicon.ico và các định dạng khác để đảm bảo hiển thị đúng trên mọi nền tảng.

Bước 1: Tạo favicon từ file logo.svg

Truy cập một trong các trang web sau để chuyển đổi file SVG thành favicon:

  • Real Favicon Generator (Khuyến nghị)
  • Favicon.io
  • CloudConvert

Tải lên file images/logo.svg và tạo các file favicon sau:

  • favicon.ico (kích thước 16x16, 32x32, 48x48)
  • apple-touch-icon.png (kích thước 180x180)
  • favicon-32x32.png
  • favicon-16x16.png

Bước 2: Lưu các file favicon

Tải xuống các file favicon đã tạo và lưu vào thư mục images/favicon/ của website.

Bước 3: Cập nhật mã HTML

Sau khi đã tạo và lưu các file favicon, hãy thêm đoạn mã sau vào phần <head> của tất cả các trang HTML (thay thế hoặc bổ sung vào thẻ favicon hiện tại):

<link rel="apple-touch-icon" sizes="180x180" href="/images/favicon/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/images/favicon/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/images/favicon/favicon-16x16.png">
<link rel="icon" href="/images/favicon/favicon.ico" type="image/x-icon">
<link rel="shortcut icon" href="/images/favicon/favicon.ico" type="image/x-icon">

Bước 4: Cập nhật robots.txt

Đảm bảo rằng Google có thể truy cập vào thư mục favicon bằng cách kiểm tra file robots.txt. Thêm dòng sau nếu cần:

Allow: /images/favicon/

Lưu ý: Sau khi thực hiện các thay đổi, có thể mất vài ngày đến vài tuần để Google cập nhật favicon trong kết quả tìm kiếm. Bạn có thể yêu cầu Google lập chỉ mục lại trang chủ của bạn bằng cách sử dụng Google Search Console.
